# Happy English Adventure

兒童英文學習網頁遊戲，適合 5–8 歲孩子練習基礎英文。

## 目前公開版本

v11 更新內容（2026-06-03）：
1. 重新命名為「測試版 (v11)」，作為後續迭代基底。
2. 移除 loading 載入畫面，開場直達首頁，減少等待感。
3. 三個功能鍵統一使用自訂圖示（Home、喇叭開/關），以 CSS 標準化大小（32×32）並加白色背景遮罩去背瑕疵。
4. 移除已無用的 IMG_HERO（跳轉圖）常量，節省約 2.1 MB。
5. 圖片集中管理為 JS 變數，避免重複內嵌，檔案體積縮小約 4MB。

v10 更新內容：
1. 首頁載入加入 loading 提示畫面，避免 13.6MB 檔案載入期間白屏。
2. 新增音效開關（header 右側 🔊/🔇 按鈕），關閉後不播放語音及音效。
3. AudioContext resume() 相容性修正，支援 Chrome 自動播放政策。
4. 測驗新增過場動畫（入場 fadeIn + 離場 slideOut），提升遊戲感。
5. 個人檔案頁面顯示「已學 X 個單元 · Y 個單字」及測驗次數。

v9 更新內容：
1. AudioContext 改為全域複用，減少資源浪費。
2. DOM 渲染優化：固定 header/page 容器，避免每次換畫面重建整個 DOM。
3. 移除練習頁面卡片右下角的喇叭圖示，點擊卡片直接播放發音，不再遮擋單字圖片。
4. 移除不再使用的 .speaker CSS 樣式。
5. 右上角頭像/名稱可點擊，直接連結到「我的檔案」頁面。

**v8｜首頁鍵圖片優化版**

## 遊戲內容

- 基礎英文單字學習
- 主題式圖卡
- 聽發音練習
- 闖關測驗
- 錯題複習
- 兒童友善介面

## 使用方式

請直接開啟 GitHub Pages 網址遊玩。

不要從 GitHub 的 `blob/main/index.html` 開啟，該頁面只是程式碼預覽，不能作為手機展示網址。
