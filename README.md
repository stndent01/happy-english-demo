# Happy English Adventure v18 - role-rewards

一個離線英文單字學習遊戲，透過圖片、發音和測驗幫助學生學習英文單字。

## 功能特色

- **圖片學習**：看圖片、聽發音，輕鬆認識英文單字
- **闖關測驗**：聽聲音選圖片 / 看圖片選聲音，兩種題型隨機出現
- **錯題練習**：自動記錄答錯的單字，提供針對性複習
- **獎勵系統**：全對可開啟寶箱，收集角色拼圖獎勵
- **多角色**：支援男生 / 女生角色，各有專屬獎勵圖
- **離線使用**：無需網路，雙擊 index.html 即可執行

## 檔案結構

```
├── index.html          # 主頁面
├── app.js              # 遊戲邏輯
├── data.js             # 單字資料 + 嵌入圖片（Base64）
├── style.css           # 樣式表
└── jszip.min.js        # ZIP 壓縮支援
```

## 使用方式

1. 雙擊 `index.html` 用瀏覽器開啟
2. 建立玩家檔案（選擇角色）
3. 選擇主題學習或直接開始測驗
4. 全對可開啟寶箱收集獎勵拼圖

## 技術細節

- 純前端實現，無需後端伺服器
- 圖片以 Base64 格式嵌入 data.js，支援離線使用
- 使用 Web Speech API 實現英文發音
- 使用 IndexedDB 保存卡片資料
- 使用 LocalStorage 保存玩家存檔
